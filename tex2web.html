TEX2WEB

(a part of partex project)

This is a web application aiming at generating a beautiful, css-styled webpage (like this) out of your .tex file (instead of traditional pdf's). Generated output will be a valid html, which you can use to publish on your website straightaway.

Just save the generated .html file and the styling file to the same directory and, always open with a browser (like chrome, firefox) to see the magic.
